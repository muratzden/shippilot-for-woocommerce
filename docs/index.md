# ShipPilot for WooCommerce

ShipPilot for WooCommerce is a WooCommerce shipping integration plugin that enables merchants to create shipments, generate shipping labels, retrieve tracking numbers, and automate shipping workflows directly from the WordPress dashboard.

## Features

- Shipment creation
- Shipping label generation
- Tracking number synchronization
- WooCommerce integration
- Secure API communication
- Translation ready
- WordPress Coding Standards compliant
